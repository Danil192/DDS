# Веб-сервис для управления движением денежных средств (ДДС)

### Основные возможности:
- Создание, редактирование и удаление записей о движении денежных средств
- Расширяемые справочники (статусы, типы, категории, подкатегории)
- Выпадающие списки (выбор категории зависит от типа, подкатегории - от категории)
- Фильтрация записей по дате, статусу, типу, категории и подкатегории

---

## Стек технологий

### Backend:
- **Python** - язык программирования
- **Django** - веб-фреймворк
- **Django REST Framework** - API для каскадных списков
- **django-filter** - фильтрация записей
- **psycopg2-binary** - драйвер для PostgreSQL

### Frontend:
- **HTML5 / CSS3** - разметка и стили
- **JavaScript** - выпадающие списки через Fetch API
- **Bootstrap** - адаптивный интерфейс

### База данных:
- **PostgreSQL** - реляционная СУБД

---

## Установка и запуск

### 1. Клонирование репозитория
git clone https://github.com/Danil192/DDS
cd DDS

### 2. Создание виртуального окружения
python -m venv venv
venv\Scripts\activate

### 3. Установка зависимостей
pip install -r requirements.txt

### 4. Запуск
python manage.py runserver
